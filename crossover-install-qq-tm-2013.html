crossover破解并安装qq tm 2013

2017.02.07 admin Linux

CrossOver 是一款可以让我们直接在 Linux 上安装和运行 Windows 软件的工具。与 wine 的区别就在于它能自动解决 Windows 程序的“依赖”问题。

1、首先到官网下载 CrossOver ，或 点此下载。

2、之后安装，再替换 winewrapper.exe.so 即可破解：

3、前往 官网 下载 qq tm 2013，安装过程见 视频教程。

长期使用下来，个人感觉，这还算比较靠谱的qq版本，大部分功能都能使用，如记住密码，传输文件，发送表情，甚至远程桌面。当然，也难免有bug，有时通知栏qq图标会消失，或者有后台有进程残留，导致无法登陆，此时终端执行 killall TM.exe ，重新登陆即可。
