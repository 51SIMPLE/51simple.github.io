BaiduPCS - 百度网盘命令行工具(不限速)

BaiduPCS 是一个多线程下载或上传百度网盘的命令行工具，可以在linux终端中使用。当然也支持Windows, Mac平台，关键在于不限速。

简单介绍一下windows下的使用方法：
1、下载 pcs.exe，并移到到个人用户目录下（如 C:\用户\Simple）
2、按Win+r快捷键，输入cmd回车，即可打开命令窗口
3、执行 pcs set --max_thread=3设置最大下载线程数
4、执行 pcs login --username=gang --password="password" ，对应自己的帐号密码进行登陆
5、之后就是一些操作文件的命令，详情见作者github主页

提示：把 pcs.exe 所在目录加入到系统的环境变量（Path）中会更方便，这样不管当前目录是什么，都能使用 pcs 命令。
