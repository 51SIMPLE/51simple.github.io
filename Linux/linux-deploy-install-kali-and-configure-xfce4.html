Linux Deploy

  • Containerization method - chroot
  • 发行版 - Kail Linux
  • 架构 - armhf
  • 发行版版本 - kali rolling
  • 安装类型 - 文件
  • 安装路径 - ${EXTERNAL_STORAGE}/kali-armhf.img
  • 镜像大小 - 自动
  • 文件系统 - ext4
  • 用户名 - android
  • 用户密码 - a
  • Privileged users - root
  • 本地化 - zh_CN.UTF-8
  • 允许挂载 Android 上的资源 - 选中
  • 挂载点 - ${EXTERNAL_STORAGE}
  • 允许 SSH 服务器启动 - 选中
  • 允许图形界面启动 - 选中
  • 图形子系统 - X11
  • 桌面环境 - Xfce

JuiceSSH

因为 Linux Deploy 已开启 SSH 服务器，我们便可以通过 JuiceSSH 去连接到它。然后就跟操作 vps 类似，只不过现在的 Linux 服务器是在安卓手机上，故连接它并不需要网络。

地址：localhost
用户名：android
密码：a
